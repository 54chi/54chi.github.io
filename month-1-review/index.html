Progress review

15 February 2016 on blog
  • CMS project: Setup publishing platform/blog (http://54chi.github.io http://54chi.com ). I like it so far. Ghost is clean and fast and I can write content offline and publish it as static pages to github, with all the benefits that this brings like: faster, cacheable, indexable, version control, database free CMS. I also created a custom theme for it, which you should be able to find in my github repo.
  • HTML5 project: CSS comics (54chi.github.io/comics http://54chi.com/tag/csscomics/ ) a fun experiment to create comics out of semantic HTML. You can find an editable demo for it here.
  • DevOps project: this one I couldn't complete because I suffered from a very slow internet speed while in Peru, however, I did a lot of research around this and found that perhaps Deis and Flynn intrigues me and I plan to use one of them (or similar) for my Elixir projects later this year.
  • Investments research: I started this year with some research on Zorro Trading, with some experimentation around automated investing with some Lite-C and R code. More to come in the next months.
  • Trip: South America: I did go to Peru. Yay me!
  • NodeJS project: Automated podcast generation with media scrapping capabilities. This is still in progress. I'm using a screen scrapper (x-ray) to fetch the contents, but I'm getting copyright regrets, so I'll probably change it to do the same but out of kickstarter campaigns instead.I didn't continue due to the low internet speed in Lima. Now that I'm back in US, I should finish it in a few more days, assuming everything is fine
